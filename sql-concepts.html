SQL Concepts

Introduction

SQL (Structured Query Language) is the standard language for interacting with relational databases. It allows users to create, read, update, and delete data in a database. SQL provides a rich set of commands and features for database management and manipulation.

Key SQL Concepts

Data Definition Language (DDL)

DDL commands are used to define and manage database structures. Examples include:

  • CREATE TABLE: Creates a new table.
  • ALTER TABLE: Modifies an existing table.
  • DROP TABLE: Deletes a table.

Data Manipulation Language (DML)

DML commands are used to manipulate data within tables. Examples include:

  • INSERT: Adds new rows to a table.
  • UPDATE: Modifies existing rows in a table.
  • DELETE: Removes rows from a table.

Data Query Language (DQL)

DQL commands are used to query and retrieve data from a database. The primary DQL command is:

  • SELECT: Retrieves data from one or more tables.

Data Control Language (DCL)

DCL commands are used to control access to data in the database. Examples include:

  • GRANT: Grants permissions to users.
  • REVOKE: Revokes permissions from users.

Transaction Control Language (TCL)

TCL commands are used to manage transactions in a database. Examples include:

  • COMMIT: Saves changes made during a transaction.
  • ROLLBACK: Reverts changes made during a transaction.
  • SAVEPOINT: Sets a point within a transaction to which changes can be rolled back.
